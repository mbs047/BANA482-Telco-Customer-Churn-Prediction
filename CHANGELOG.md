# Changelog

## v1.0.0 - Initial Case 1 Release

This is the first public release of **BANA482 | Telco Customer Churn Prediction**.

### Included

- Reproducible R Markdown analysis for Telco customer churn prediction.
- Raw Telco Customer Churn dataset used in the project.
- Final written report in PDF and Word formats.
- Final presentation slides in PDF and PowerPoint formats.
- Rendered code report outputs.
- Saved model evaluation figures for churn insights and ROC/AUC.
- Repository documentation, licensing, and public hygiene setup.

### Notes

- Generated model files and prediction exports are intentionally excluded from version control and can be regenerated from the R Markdown analysis.
- Class-provided reference PDFs remain local and are not relicensed in this public repository.
